This function returns a VDOM Element with given arguments.

| param | type |
|-------|------|
| tagName | String |
| ...arguments | Array/String/VDOM|

***Example***
``` 
Power.h('h1', 'Hello World', {class: 'myClass},
  Power.h('h2', 'I am a child of Hello World')
)
```
